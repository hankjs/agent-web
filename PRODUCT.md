# Product

## Register

brand

## Users

General users (普通用户) interacting with an AI agent through a desktop application. Not developers or power users; everyday people who want AI assistance without needing to understand the underlying mechanics. They expect clarity, confidence, and a sense of quality from the interface itself.

## Product Purpose

A desktop AI agent that executes tasks autonomously. The interface is the user's window into what the agent is doing: thinking, acting, completing. Success looks like the user feeling in control and informed without being overwhelmed by technical detail.

## Brand Personality

Bold, distinctive, minimal. The interface should feel like a precision instrument: confident in its choices, quiet where it doesn't need to speak, striking where it does. Not warm and fuzzy; not cold and technical. Somewhere between a well-designed camera and a beautifully typeset book.

## Anti-references

- Generic chatbot UIs (ChatGPT, Claude web): bubble layouts, alternating message rows, "how can I help you today" emptiness
- Terminal / hacker aesthetic: monospace everything, green-on-black, raw output dumps
- Corporate SaaS: metric cards, sidebar navigation trees, breadcrumb trails

## Design Principles

1. **Content is the interface.** The agent's work IS the visual. Don't decorate around it; let it breathe and command attention on its own.
2. **Confidence through restraint.** Every element earns its place. If removing something doesn't hurt, it was never needed.
3. **Show the machine, not the chat.** This isn't a conversation; it's an agent working. The UI should reflect execution and progress, not dialogue.
4. **Distinctive at a glance.** Someone seeing this for 2 seconds should not confuse it with any other AI product.
5. **Respect the user's attention.** Information appears when relevant, disappears when not. No persistent clutter.

## Accessibility & Inclusion

WCAG AA compliance. Standard contrast ratios, full keyboard navigation, screen reader support. Reduced motion preference respected.
